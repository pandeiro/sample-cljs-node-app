## sample-cljs-node-app

Based on the [macchiato](https://macchiato-framework.github.io/) template

### Prequisites

- [Node.js](https://nodejs.org/en/)
- [Leiningen](https://leiningen.org/)

---

### Development

Using two terminals:

```sh
$ lein build
```

And

```sh
$ npm start
```

---

### Release

```sh
$ lein package
$ npm start
```
